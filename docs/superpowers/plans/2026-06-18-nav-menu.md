# Nav Menu (Hamburger) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the repeated "Ajustes" / "Historial" / "Salir" links in four navbars with a single self-contained `NavMenu` hamburger-icon component.

**Architecture:** One new client component, `src/components/NavMenu.tsx`, with no props — it manages its own open/closed state, closes on outside-click or Escape, and calls `authClient.signOut()` + `router.push('/')` directly. It's dropped into four existing navbar files, replacing the old links/button and the per-page sign-out plumbing that becomes dead code as a result.

**Tech Stack:** React (client components), Next.js App Router (`next/navigation`, `next/link`), existing `authClient` from `src/lib/auth-client.ts`. No new dependencies — the hamburger icon is an inline SVG.

**Spec:** `docs/superpowers/specs/2026-06-18-nav-menu-design.md`

## Task 1: Create the `NavMenu` component

**Files:**
- Create: `src/components/NavMenu.tsx`

- [ ] **Step 1: Create the file**

```tsx
'use client'

import { useEffect, useRef, useState } from 'react'
import { useRouter } from 'next/navigation'
import Link from 'next/link'
import { authClient } from '@/lib/auth-client'

export function NavMenu() {
  const router = useRouter()
  const [open, setOpen] = useState(false)
  const containerRef = useRef<HTMLDivElement>(null)

  useEffect(() => {
    if (!open) return

    const handleClickOutside = (e: MouseEvent) => {
      if (containerRef.current && !containerRef.current.contains(e.target as Node)) {
        setOpen(false)
      }
    }
    const handleEscape = (e: KeyboardEvent) => {
      if (e.key === 'Escape') setOpen(false)
    }

    document.addEventListener('mousedown', handleClickOutside)
    document.addEventListener('keydown', handleEscape)
    return () => {
      document.removeEventListener('mousedown', handleClickOutside)
      document.removeEventListener('keydown', handleEscape)
    }
  }, [open])

  const handleSignOut = () => {
    setOpen(false)
    authClient.signOut().then(() => router.push('/'))
  }

  return (
    <div className="relative" ref={containerRef}>
      <button
        onClick={() => setOpen((v) => !v)}
        aria-label="Menú"
        aria-expanded={open}
        className="flex items-center justify-center h-8 w-8 rounded-md border border-border text-muted-foreground hover:text-foreground hover:border-foreground/30 transition-colors"
      >
        <svg width="16" height="16" viewBox="0 0 16 16" fill="none" stroke="currentColor" strokeWidth="1.5" strokeLinecap="round">
          <line x1="2" y1="4" x2="14" y2="4" />
          <line x1="2" y1="8" x2="14" y2="8" />
          <line x1="2" y1="12" x2="14" y2="12" />
        </svg>
      </button>

      {open && (
        <div className="absolute right-0 top-full mt-2 w-44 rounded-lg border border-border bg-background shadow-lg p-1 z-50">
          <Link
            href="/ajustes"
            onClick={() => setOpen(false)}
            className="block px-3 py-2 rounded-md text-xs text-muted-foreground hover:text-foreground hover:bg-muted transition-colors"
          >
            Ajustes
          </Link>
          <Link
            href="/historial"
            onClick={() => setOpen(false)}
            className="block px-3 py-2 rounded-md text-xs text-muted-foreground hover:text-foreground hover:bg-muted transition-colors"
          >
            Historial
          </Link>
          <div className="h-px bg-border my-1" />
          <button
            onClick={handleSignOut}
            className="block w-full text-left px-3 py-2 rounded-md text-xs text-rose-400 hover:bg-muted transition-colors"
          >
            Cerrar sesión
          </button>
        </div>
      )}
    </div>
  )
}
```
